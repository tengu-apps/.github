# Tengu

Self-hosted PaaS for deploying web applications via git push.

```bash
git push tengu main
```

## Repositories

| Repo | Description |
|------|-------------|
| [tengu](https://github.com/tengu-apps/tengu) | Core PaaS server (Rust) |
| [tengu-init](https://github.com/tengu-apps/tengu-init) | Hetzner Cloud provisioner |
| [tengu-caddy](https://github.com/tengu-apps/tengu-caddy) | Custom Caddy with Cloudflare DNS |
| [tengu-deb](https://github.com/tengu-apps/tengu-deb) | Debian package repository |
| [tengu-desktop](https://github.com/tengu-apps/tengu-desktop) | Desktop GUI (Dioxus) |
| [tengu-website](https://github.com/tengu-apps/tengu-website) | Landing page |
